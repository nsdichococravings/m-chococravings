1
Create Your Supabase Account
Supabase is a free cloud database. Your order data will be stored here securely.
1
Go to supabase.com and click "Start your project"
2
Sign up with your Google account (fastest) or email
3
Click "New Project" → Name it "nsdi-chococravings"
4
Set a Database Password (save this somewhere!) → Choose Region: Mumbai (ap-south-1) → Click Create Project
5
Wait ~2 minutes for project to be ready
✅
Free forever: Supabase free tier gives you 500MB storage and 2GB data transfer per month — more than enough to handle thousands of orders!
2
Get Your API Keys
These keys connect the app and admin panel to your Supabase database.
1
In your Supabase project, click Settings (gear icon) in the left sidebar
2
Click API under Project Settings
3
Copy your Project URL — looks like: https://abcdefghij.supabase.co
4
Copy the anon / public key (the long string starting with eyJhbGci…)
⚠️
Important: Use only the anon / public key in the app. Never use the "service_role" secret key in frontend code — that's only for server-side use.
Now paste your keys into BOTH files:
📱 chococravings-app-v2.html — find line ~30
// Replace these two lines:
const SUPABASE_URL = 'https://YOUR_PROJECT_ID.supabase.co';
const SUPABASE_ANON_KEY = 'YOUR_ANON_PUBLIC_KEY';

// With your actual keys like this:
const SUPABASE_URL = 'https://abcdefghij.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
📊 chococravings-admin-v2.html — find line ~30
// Same keys, replace in admin file too:
let SUPABASE_URL = localStorage.getItem('sb_url') || 'https://abcdefghij.supabase.co';
let SUPABASE_KEY = localStorage.getItem('sb_key') || 'eyJhbGciOiJIUzI1NiIs...';
💡
Shortcut: In the Admin Panel, go to Settings tab and paste your keys there — they'll be saved automatically to your browser!
3
Create the Database Tables
Run the SQL script to create your orders, customers, and notifications tables.
1
In Supabase, click SQL Editor in the left sidebar
2
Click "New Query"
3
Open the file supabase-setup.sql (provided) → copy its entire contents → paste into the SQL editor
4
Click the green ▶ Run button → you should see "Success. No rows returned"
Tables that will be created
orders              → stores every customer order
order_status_history → tracks status changes over time
customers           → customer profiles + loyalty points
notifications       → log of all WhatsApp/email alerts sent
✅
Verify it worked: Click Table Editor in the left sidebar — you should see the 4 tables listed there.
4
Enable Realtime (Live Order Feed)
This makes new orders appear instantly in the admin panel without refreshing.
1
In Supabase, go to Database → Replication in the left sidebar
2
Find the orders table in the list
3
Toggle ON for INSERT and UPDATE events
4
Click Save
🔔
What this does: Every time a customer places an order, the admin dashboard will automatically update and play a notification sound — just like Swiggy/Zomato!
5

4
You receive the order details on +91 99721 41890 — your WhatsApp Business number
5
From the Admin Panel, click 📲 button on any order to send status updates to the customer
