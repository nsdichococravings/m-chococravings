Browser

2
A panel opens — fill in these details:

App Version:     1.0.0
Version Code:    1
Display Mode:    Standalone
Orientation:     Portrait
Theme Color:     #6B2FCE

3

4
Download the ZIP file → inside you'll find app-release-bundle.aab and a .keystore file
🔐
SAVE THE KEYSTORE FILE! PWABuilder generates a signing key for you. Download it and save to Google Drive immediately. You need it for every future update — if lost, you can never update the app.
📋
Also save: The key passwords shown on screen — copy them to Notes or Google Keep before closing.

Apple App Store — Needs a Mac
iOS apps must be built and signed using Xcode, which only runs on macOS. This cannot be done from a Samsung Android phone. Your options:
💻
Borrow a Mac friend's laptop — Install Xcode, run all iOS steps in our publishing guide in ~2 hours
☁️
MacInCloud.com — Rent a cloud Mac for ~₹80/hour. Do the iOS build in 2 hours = ₹160 total
📱
Launch Android first — Get users on Play Store first. Add iOS later when you have access to a Mac. Most Indian users are Android anyway!
